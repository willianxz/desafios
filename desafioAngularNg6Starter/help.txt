FAÇA O DOWNLOAD DO PROJETO NO GITHUB.

1 - A pasta Arquivos contem o projeto mais enxuto no formato html, sem o uso do Angular cli, não sendo necessario nenhum uso do NPM, basta clicar no arquivo index.html

2 - A pasta testado-app contem o projeto utilizando em conjunto com o framework Angular. Para instalar ele, você terá que abrir o CMD(terminal de comando) no diretorio raiz onde está os arquivos do projeto, digitando o seguinte comando:
npm install 
Esse comando irá instalar as dependencias do projeto, instalando a pasta node_modules para que o mesmo funcione corretamente.
Em seguida, você deve usar o comando:
ng serve --open
Dessa forma, irá rodar o projeto pelo navegador.

Para trocar a imagem, sobre o editor dê um duplo clique com o mouse em cima da mesma ou use o icone de imagem para adicionar alguma imagem no editor.

SOBRE:
O quillsjs é uma biblioteca em Javascript para incluir um editor online de textos em html, semelhante ao editor do Wordpress.


OBS: Você não pode adicionar eventos de click de javascript em elementos html que estão dentro da div do editor. Para contornar isso existe dois modos para aplicar um evento ex1:

quill.root.addEventListener('dblclick', (e) => {
  const img = Parchment.find(e.target);
  if (img instanceof ImageBlot) { //Verificamos se o que foi clicado é uma imagem.
    //Se você colocar esse evento abaixo fora do IF,ao invés de ter que clicar em uma imagem para trocar, você poderá clicar em qualquer parte do editor 
	//E colocar uma imagem.
    document.querySelector(".ql-image").click(); 
    quill.setSelection(img.offset(quill.scroll), 1, 'user');
  }
});


ex2:
Você pode colocar qualquer evento em um elemento html fóra da div do editor, um botão por exemplo e a partir desse ponto fazer algo.


O quillsjs, não se da bem com imagens locais dentro do editor, por esse motivo, é necessario reproduzir a  mesma usando na tag html "img" em formato BASE64 ou então que a imagem esteja online, sendo passado a URL dela.

Dentro da div do editor, você não pode colocar nenhum ID, nas tags html, pois o quillsjs não aceita.

Algumas formatações CSS são ignoradas em algumas tags html que estão dentro da div do editor.
